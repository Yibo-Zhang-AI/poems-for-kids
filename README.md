# 🌸 小小诗人学古诗

一个适合6岁孩子学习中国古诗词的网站。

## 📖 包含的古诗

1. **黄鹤楼** - 崔颢
2. **忆江南** - 白居易
3. **寻隐者不遇** - 贾岛
4. **龟虽寿** - 曹操
5. **相思** - 王维

## ✨ 特色

- 🎨 彩色卡片设计，吸引孩子注意力
- 🖼️ 每首诗配一张精美图片
- 💬 用孩子能听懂的话解释诗意
- 🎈 有趣的小知识
- 📱 支持手机、平板、电脑访问

## 🌐 在线访问

访问地址：https://[你的用户名].github.io/poems-for-kids

## 🚀 本地运行

```bash
# 使用 Node.js
node server.js

# 或使用 Python
python -m http.server 8080
```

然后打开 http://localhost:8080

## 📄 文件说明

- `index.html` - 主页面
- `server.js` - 本地服务器（可选）
- `package.json` - 项目配置

## 📝 使用说明

1. 点击目录中的诗名跳转到对应诗歌
2. 每首诗都有：
   - 原文（大字显示）
   - 配图
   - 白话解释
   - 有趣小知识
3. 点击右下角的 ⬆️ 返回顶部

## 🎨 技术栈

- HTML5
- CSS3 (Flexbox, Gradient)
- 响应式设计
